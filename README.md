# Magnetometer
Программа распознавания металлических предметов для подводных аппаратов. 

Программа предназначена для проверки и калибровки магнитометра PNI TCM 2.5, а также для получения данных, записи их в файлы и обмена данными с технологической программой одного из модулей изделия. Реализвана запись в бинарные файлы для дальнейшего использования. Формируется отчет по результатам работы. Реализован алгоритм компенсации внешних магнитных полей.
